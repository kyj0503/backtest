# 백엔드 테스트 문서

이 디렉토리는 백엔드 애플리케이션의 테스트 전략, 구조, 실행 방법에 대한 문서를 포함합니다.

## 문서 목록

-   [**테스트 전략 (`strategy.md`)**](./strategy.md): 프로젝트의 테스트 철학과 각 테스트 레이어(단위, 통합, E2E)의 역할에 대해 설명합니다.
-   [**테스트 실행 및 환경 (`execution.md`)**](./execution.md): `pytest`를 사용하여 테스트를 실행하는 방법, 마커 사용법, 테스트 환경 설정에 대해 안내합니다.
-   [**픽스처 및 테스트 데이터 (`fixtures.md`)**](./fixtures.md): 테스트에서 사용하는 `pytest` 픽스처와 테스트 데이터 생성 방법에 대해 설명합니다.
-   [**비동기 코드 테스트 (`async.md`)**](./async.md): `pytest-asyncio`를 활용하여 비동기 함수 및 서비스를 테스트하는 방법에 대해 다룹니다.
