# 프론트엔드 테스트 문서

이 디렉토리는 프론트엔드 애플리케이션의 테스트 전략, 도구, 작성 방법에 대한 문서를 포함합니다.

## 문서 목록

-   [**테스트 전략 (`strategy.md`)**](./strategy.md): 프론트엔드 테스트의 목표와 각 테스트 레벨(단위, 컴포넌트, E2E)의 역할을 설명합니다.
-   [**테스트 환경 및 실행 (`execution.md`)**](./execution.md): Vitest와 React Testing Library를 사용한 테스트 환경 설정 및 실행 방법을 안내합니다.
-   [**컴포넌트 테스트 (`component.md`)**](./component.md): React 컴포넌트를 테스트하는 방법, 사용자 상호작용 시뮬레이션, 상태 변화 검증에 대해 다룹니다.
-   [**커스텀 훅 및 리듀서 테스트 (`hooks.md`)**](./hooks.md): `useReducer`로 작성된 복잡한 로직이나 커스텀 훅을 테스트하는 방법을 설명합니다.
